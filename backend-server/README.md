# Backend Server (Express + SQLite)

## Setup
```bash
cd backend-server
npm install
npm run seed   # create teams + users tables and sample data
npm run dev    # http://localhost:3001
```

- ENV: see `.env`
- DB path (Windows safe): we resolve to `../database/sqlite/wallboard.db` relative to backend.
